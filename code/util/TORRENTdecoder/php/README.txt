TorrentEditor.com API - Simple API to edit torrents
Version: 0.1 beta

Please link to torrenteditor.com if you use this API in your project.

--REQUIREMENTS--
(1) PHP 5

--HOW TO--
See examples for how to use the API.

The process generally consists of four steps:
  (1) Get the torrent file contents from a local file or via upload
  (2) Initialize a torrent object with the data
  (3) Read/modify the torrent through the API
  (4) If needed, save the modified torrent
